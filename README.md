# StatusSystem
Minecraft Java Edition 1.21.10 Vanillaでのみ動作を確認しています。(Data Pack Version 88)

マクロのあるバージョンならだいたい動くと思いますが、上記環境以外での動作は保証しません。

また、別途[OhMyDat](https://github.com/Ai-Akaishi/OhMyDat)を導入する必要があります。
